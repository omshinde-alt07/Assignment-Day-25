# 🤖 Types of Machine Learning

## Prompt
Explain types of machine learning with real-world examples.

---

## Output

Machine Learning is a field of artificial intelligence where systems learn from data and improve their performance without being explicitly programmed. There are mainly three types of machine learning:

---

### 1. Supervised Learning

Supervised learning is a type of machine learning where the model is trained on labeled data. This means the input data is paired with the correct output.

- **Example:**  
  Predicting house prices using features like area, location, and number of rooms.  
  Detecting whether an email is spam or not.

- **Real-world use case:**  
  Used in email filtering, credit scoring, and sales prediction.

---

### 2. Unsupervised Learning

Unsupervised learning works with unlabeled data. The model tries to find patterns or groupings in the data.

- **Example:**  
  Grouping customers based on their purchasing behavior.  

- **Real-world use case:**  
  Used in customer segmentation, market analysis, and recommendation systems.

---

### 3. Reinforcement Learning

Reinforcement learning is a type of learning where an agent learns by interacting with an environment and receiving rewards or penalties.

- **Example:**  
  A robot learning to walk by trying different actions and getting rewards when it succeeds.  

- **Real-world use case:**  
  Used in robotics, gaming (like chess or video games), and self-driving cars.

---
